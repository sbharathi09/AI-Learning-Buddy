# 🌱 AI Learning Buddy

An AI-powered learning assistant built with **Python**, **Streamlit**, **Google Gemini AI**, and the **Wikipedia API**. It provides personalized learning experiences for students from **Kindergarten to Postgraduate** by generating explanations, notes, quizzes, summaries, coding problems, and more.

## 🚀 Live Demo

🔗 **Live App:** https://ai-learning-buddy-ymnwvjbldvym2akdvt8pbg.streamlit.app/

## ✨ Features

* 📚 Supports learners from Kindergarten to Postgraduate
* 🎯 Personalized content based on education level and subject
* 🤖 AI-powered explanations using Google Gemini
* 🌍 Wikipedia summaries and topic images
* 📝 Notes, summaries, and 5-mark/10-mark answers
* ❓ MCQs, quizzes, and practice questions
* 💻 Coding problems and interview questions
* 📥 Download generated notes
* 💬 Session-based chat history

## 🛠️ Tech Stack

* Python
* Streamlit
* Google Gemini API
* Wikipedia API

## 📦 Installation

```bash
git clone https://github.com/sbharathi09/AI-Learning-Buddy.git
cd AI-Learning-Buddy
pip install -r requirements.txt
streamlit run app.py
```

## 🔑 Configuration

Create a `.streamlit/secrets.toml` file and add:

```toml
GEMINI_API_KEY = "YOUR_GEMINI_API_KEY"
```
